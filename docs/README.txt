# Bogoshop Documentation YAAAAAY

Welcome to Bogoshop. Press UP/DOWN to scroll. Press H or ESC to close.
Btw. Bogo not in like "buy one get one free", it's bogo as in "bogos brinted?... Zirp Zorp"

I've created this program out of pure ANGER... I HATE photoshop and its maaaaanyyyy features.
I like my workspace nice and clean. Not too many functions right in my face... rather just the
essential ones that ACTUALLY work.

## Navigation

- Arrow Keys    Pan the image (when zoomed or in mirror mode)
- +  /  -       Zoom in / out (min 1.0x unless mirror mode)
- R             Reset zoom, pan and mirror mode
- M             Toggle mirror mode
- F             Toggle fullscreen
- H             Open this help file
- Q / ESC       Quit (asks to save first)

## Effects

Enter a number and press ENTER to apply an effect.
Type  <number>H  to open the documentation for that effect.

- 1             Brighten
- 2             Darken
- 3             Icing
- 4             Negative

## Cropping

Type a ratio (e.g. 4:3 or 1:1) and press ENTER to crop.

## Workflow

1. Zoom and pan to frame your image
2. Crop to desired ratio
3. Press C to composite (freeze viewport as new image)
4. Press S to save or E to export

## Save / Export

- S             Save (overwrites original, asks Y/N first)
- E             Export (opens Windows save dialog)

## Background

- T             Randomize background color

## Composite

- C             Freeze current viewport as new base image.
                Mirror mode is deactivated after compositing.
